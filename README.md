# Deloosional

A detective-themed investigation board that gamifies studying. Transform concepts into clues, connect ideas with red strings, and solve the mystery of any subject. Built with vanilla JavaScriptâ€”no backend, no frameworks, just pure client-side magic.

## Overview

Deloosional turns studying into an engaging investigation. Students create visual crime boards with sticky notes, images, and connections, making learning more interactive and memorable than traditional note-taking. The detective theme and game-like mechanics motivate exploration of relationships between concepts.

## Features

- **Multi-Board System**: Up to 6 boards with previews, rename, delete, and export
- **Sticky Notes**: Four types (Concept, Fact, Question, Theory) with custom colors, drag-and-drop, and inline editing
- **Media Attachments**: Image uploads with annotation/highlighting tools
- **Red String Connections**: Visual links between notes and media with toggleable visibility
- **Zoom & Pan**: Navigate large boards with 0.25x-3x zoom and mouse drag
- **Board Export**: High-resolution PNG exports
- **Case Closed Animation**: Satisfying completion feedback with sound effects
- **Welcome Page**: Board management with animated titles and auto-generated previews

## Tech Stack

Vanilla JavaScript, HTML5 Canvas, CSS3, localStorage, html2canvas (CDN). Client-side rendering with no build process required.

## Requirements

Modern browser (ES6+, Canvas API, localStorage, File API). Optional: Stack Sans Headline font.
